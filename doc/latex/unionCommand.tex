\doxysubsection{Detailed Description}
Make all command types the same size and interchangable. 

This is useful for arrays or making a common entry point for all command types. The exact type information can be recovered later with the {\itshape \mbox{\hyperlink{command_8c_a1708aa65874060aa7ef99bf9e1671f5c}{get\+\_\+command\+\_\+type()}}} function.
